1.  [22,27,16,2,18,6] -> Insertion Sort. Dizinin sort türüne göre aşamalarını yazınız.

- [22,27,16,2,18,6]  n
- [16, 22, 27, 2, 18, 6] n-1
- [2, 16, 22, 27, 18, 6] n-2
- [2, 16, 18, 22, 27, 6] n-3
- [2, 6, 16, 18, 22, 27] n-4

2.  O(n^2)
3.  Best Case   : O(n)  
    Average Case: O(n^2)  
    Worst Case  : O(n^2)  
4. 18 -> Averga Case O(n^2)
